# 🌿 สรุปโครงการ AgroAi (GitHub: codxvoid/AgroAi)
**หัวข้อ:** ระบบอัจฉริยะแบบ Multi-agent เพื่อการเกษตรสมัยใหม่

---

## 📋 ข้อมูลทั่วไป (Overview)
**AgroAi** เป็นระบบที่ออกแบบมาเพื่อเพิ่มประสิทธิภาพในการทำเกษตรกรรมโดยใช้เทคโนโลยี AI และข้อมูลแบบ Real-time เพื่อช่วยในการตัดสินใจและคาดการณ์ผลผลิต

*   **ผู้พัฒนา:** Keshari Nandan ([Codxvoid](https://github.com/codxvoid))
*   **เว็บไซต์โครงการ:** [agroai.pages.dev](http://agroai.pages.dev)
*   **เทคโนโลยีที่ใช้ (Tech Stack):** 
    *   JavaScript (React/Vite) - 92.4%
    *   HTML & CSS (Tailwind CSS) - 7.6%
    *   Infrastructure: AI-LLM Agents

---

## ✨ คุณสมบัติหลัก (Core Features)
1.  **Geolocation & Weather Integration:** เชื่อมต่อข้อมูลตำแหน่งและสภาพอากาศแบบปัจจุบัน
2.  **Climate Impact Modeling:** จำลองผลกระทบของอุณหภูมิและสภาพภูมิอากาศต่อการเติบโตของพืช
3.  **Chemical Usage Tracking:** ติดตามการใช้สารเคมีและวิเคราะห์ผลกระทบต่อสุขภาพมนุษย์
4.  **Crop Suitability Forecasting:** พยากรณ์ความเหมาะสมของพืชในแต่ละพื้นที่ โดยพิจารณาจากปัจจัยรอบด้าน (อากาศ, ดิน, ประวัติผลผลิต)
5.  **Terminal / CLI AI Query:** ระบบสอบถามข้อมูลผ่าน Command Line ที่สามารถทำนาย อธิบาย และให้คำแนะนำได้

## 💡 ตัวอย่างการใช้งาน (Use Case)
*"เดือนนี้ในจังหวัดนี้ควรปลูกอะไรดีที่สุด เมื่อพิจารณาจากอากาศปัจจุบัน และประวัติการใช้ปุ๋ย?"*
AgroAi จะทำการ:
*   วิเคราะห์ตำแหน่งและสภาพอากาศ
*   ประเมินสภาพดินจากข้อมูลทางภูมิศาสตร์
*   ติดตามประวัติการใช้สารเคมี
*   ประเมินผลกระทบต่อสุขภาพจากสารตกค้าง
*   แนะนำพืชที่เหมาะสมที่สุดพร้อมเหตุผลประกอบ
